db_errormessage

Name

db_errormessage  --  Returns the last error string from the database or a given result.

Synopsis

        $message = db_errormessage($result=null);

Description

Occasionally, the database layer will suffer an error of some sort: perhaps bad SQL was sent in, or it hit a bug, or some internal limit was reached, or the disk filled up. These will result in some functions returning false that normally don't.

This function returns the last error message string produced by the database layer. You can pass in a result resource (such as returned by db_send ) if you want to know specifically what happened on the corresponding query.
