db_insert_hash

Name

db_insert_hash  --  Inserts a new row into a table.

Synopsis

        $pg_result = db_insert_hash($table, $hash)

Description

This function inserts a hash (of key/value pairs) into the given table. The keys are treated as column names, and the values as items to be inserted. The value returned is a Postgres result resource.

This is most useful when the array is $_POST, the results of a form submission, with information that you want to stuff into the database.

$array = array('first_name' => 'Steve', 'user_id' => 5);
db_insert_hash('users', $array);
