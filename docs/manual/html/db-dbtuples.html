db_tuples

Name

db_tuples  --  Returns the number of rows affected by the given result.

Synopsis

        $pg_result = db_send(...);
        $count = db_tuples($pg_result);

Description

Given a result resource returned by db_send, this returns the number of rows affected by the query.
