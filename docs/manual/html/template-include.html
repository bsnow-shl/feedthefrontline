template_include

Name

template_include -- Searches backwards from the current file, looking for a file to include, and include it

Synopsis

    void template_include(string filename, bool current_dir_only=false);

Description

Searches backwards through the directory tree from the current part looking for the given filename, and if it's found, streams it out to the browser. If the file is not found, nothing is output and the function returns.

filename is the filename to go looking for, and current_dir_only is a boolean indicating whether the templating system should only search the current directory. This function returns nothing of interest.

The part being included is run through the templating system in the usual fashion: first its .data counterpart is run, then the file itself. The file may contain PHP code that is run in the usual way.
