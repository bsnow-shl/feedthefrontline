db_newid

Name

db_newid  --  Obtains a new number out of a sequence.

Synopsis

        $id = db_newid(identifier)

Description

Frequently, database systems need an auto-incremementing counter to generate primary keys. Every database vendor provides you a different one, and this function provides an abstraction over them all.

identifier is the name of the identifier that you are counting through. This function returns a number 1 greater than all previous return values for identifier, or 1 if this is the first call for the given identifier.

$id = db_newid('users');
db_send("insert into users (user_id) values (?)", $id);

The new ID is guaranteed to be unique compared to all other ID's generated for that identifier, even if the system is under load.
