template_disable

Name

template_disable -- Turns off templating for the duration of this web request

Synopsis

    void template_disable();

Description

There may be occasions when you don't want your _template file to be run while serving a particular web hit. This is sometimes done while serving pages meant to be printed out, or when you don't want to distract the visitor with unnecessary links to go other places, such as while completing an eCommerce order.

When placed in your part's .data file, a call to template_disable() will prevent the _template.html file from being used while serving the page. Your part will have to provide all the ceremonial material of a web page: HTML, HEAD, BODY, and so on.
