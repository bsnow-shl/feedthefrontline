Questions and answers about system translations

System translations contain text that is generated by system software and plug-in modules. These generate user-visible text, which needs to be translated.

As a result, you will often see little bits of confusing markup in the original English. You should very closely follow the formatting and layout of the original English when performing a translation.

HTML markup

Sometimes you will see HTML markup like <b> and <i> or hyperlinks like

Click <a href="file.html">here.

Reproduce this material exactly, including the HTML markup. A good way to start is to copy and paste the English, then replace all the English words with the corresponding phrases in the corresponding language.

Formatting strings

Sometimes you will see formatting strings like

Your order number is %s.
or
%s out of %s points obtained
. In these cases the %s are substituted with variables by the underlying software module.

When translating this material, your final translation should have as many % markers as in the original text, and they should have the same semantic meaning given their position in the text. For instance, in the above example of "%s out of %s points obtained" (as in, "5 out of 10"), your translation should have the first %s signify the number of points obtained, and have the second %s signify the number that could be obtained.

Example

Original text is:
"You got %s points out of a possible %s."
Good translation:
"Vous avez obtenu %d points du maximum de %d."
Bad translation:
"Du maximum de %d points, vous avez obtenu %d".
This is bad because it reverses the meaning of the % substitutions.
